ls(list)- выводит список содержимого в папке(включая файлы и папки)

cd(change directory) - переход между папками

cd../ - переход на одну папку назад

ctrl + l (+ clear) - очистка терминала

pwd(path working directory) - абсолютный путь к папке в которой вы находитесь

ls -a - показать содержимое папки (включая скрытые папки и файлы)

rm -rf .git
git status

git init- инициализация git репозитория

git remote add origin <SSH-url> - привязка локального репозитория к удаленному 

git remote -v - посмотреть к какому локальному репозиторию привязан
